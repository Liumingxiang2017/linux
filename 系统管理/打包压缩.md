# 压缩与解压缩

## gzip 

只能压缩单个文件,结果为.gz

- -1 快速压缩 -9 最佳压缩
- -l 查看详细信息
- 解压缩为 gunzip 或者 gzip -d

## zip

- -r 压缩目录
- 解压缩为unzip

## tar 打包并压缩

- c 创建压缩文件 Create
- z 指定类型为 gzip
- j 指定类型为 bzip2
- v 显示压缩过程 Verbose
- f 指定打包压缩后的文件名 File
- tvf 以列表显示里面的内容
- zcvf 格式：tar zxvf directory.tar.gz directory打包并压缩为.tar.gz
- zxvf 解压缩

红帽在rhel7中tar可以检测压缩包类型，不需要指定z,j
